Simulate geographic restrictions and display compliance details.
this tool exposes legislature contact information and addresses the persons responsible for the existence of the policy that is preventing internet access that should otherwise be free from legislation. 

this repository is for civic awareness purposes only and does not condone illegal activity.

in order to use this repository you need only to include this line in your html:

```<script src="https://argonianvoter.github.io/FightOSA/OSA_georejector.js"></script>```

it should be noted that this repository can assist in legal compliance but cannot guaruntee it.
